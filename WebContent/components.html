Components

  • repeson4bchn

    A JSON-RPC client for the Bitcoin Cash Node

    This components implements asynchronous JSON-RPC calls to the node. It can be used to verify signatures, validate addresses, or sending coins to addresses.

  • cashid4spring

    CashId Authentication for Spring Security.

    It uses the repeson4bchn component to process authentication requests.

  • Muro.Cash

    The main component.

    It includes the classes that make it possible to build a Paywall by just defining some properties and by annotating methods.

  • Web Extension

    A Web Extension that works as an CashId authentication provider, and as a Wallet with which contents can be bought.

    At the moment, Muro.Cash uses Bitcoin.com's Badger Wallet.
